You're working in a healthcare startup building a compression system for chest X-ray images using autoencoders. Your team has proposed the following 4 architectures for evaluation. Each diagram shows the number of neurons in the input layer, hidden layer, and output layer. Your job is to choose the architecture(s) that will best compress the input while still allowing reasonable reconstruction. Based on the diagrams shown, identify which architecture(s) represent an undercomplete autoencoder.

Answer: A and C
Explanation: An undercomplete autoencoder has a hidden layer smaller than the input layer, forcing the network to learn a compressed representation. Architectures A and C have hidden layers smaller than their input layers, making them undercomplete.

Which of the following best describes the role of the hidden representation in this autoencoder setup?

Answer: It captures a lower or higher-dimensional encoding useful for reconstruction.
3

What can be said about Model A's architecture in this scenario?

Answer: It is an undercomplete autoencoder.
It compresses the input data to extract meaningful features.

What is the major risk of using Model B in this situation?

Answer: It could trivially copy input to output without learning meaningful representations.

Suppose the autoencoder trained on normal sensor data starts showing high reconstruction error on new input data. What does this indicate?

Answer: The new input is likely anomalous or different from training distribution.

Why would using squared error loss with sigmoid decoder and binary input be suboptimal?

Answer: It assumes Gaussian distribution, which mismatches binary input assumptions.
9

Which decoder activation function is most suitable for the given input scenario?

Answer: linear
10

Suppose the inputs are normalized but still real-valued. Which loss function best suits the model's objective?

Answer: L(θ)= ∑(x̂ij − xij)²
11

Why is binary cross-entropy loss generally not appropriate when training an autoencoder on real-valued input data?

Answer: It assumes binary targets in[0,1], which doesn’t match real-valued ground truth.
12

Under the above conditions, what can be said about the representations learned by both methods?

Answer: PCA and the linear autoencoder will learn identical subspaces
13

If the dataset is found to lie on a nonlinear manifold, which approach would likely perform better for compression and why?

Answer: Nonlinear autoencoder, because it can learn nonlinear mappings
14

The team now wants to visualize the compressed 2D representation of the data. Researcher A uses PCA with 2 components, while Researcher B trains a nonlinear autoencoder with a 2- dimensional hidden layer. Which of the following is true?

Answer: The autoencoder may learn better separability due to nonlinearity
15

Why is this trivial copying behavior a problem in overcomplete autoencoders?

Answer: It results in representations that don't generalize

Answer: Adding noise to inputs
Penalizing activations to enforce sparsity
Penalizing the Jacobian of hidden activations

Which of the following statements correctly compares the regular and denoising autoencoder approaches for this medical image task?

Answer: Denoising autoencoder explicitly learns to remove noise and is better suited for this application
20

An engineer builds an autoencoder for real-valued inputs but uses a sigmoid decoder activation and binary cross-entropy loss. What issue might arise from this setup?

Answer: There will be a mismatch between output range and data distribution

Answer: L(θ)= ∑∥xi − x̂i∥² + λ∑KL(ρ∥ρ̂j)

What is the main idea behind a Sparse Autoencoder?

Answer: It restricts most hidden units to be inactive(close to 0)
23

Which activation pattern is most likely from a well-trained sparse autoencoder?

Answer: Only a small subset of hidden units fire for each input
24

What is the key regularization objective of a Contractive Autoencoder?

Answer: Penalize the sensitivity of the encoder to small input changes

In a Contractive Autoencoder, the added penalty on the Jacobian of hidden activations with respect to the input encourages which of the following behaviors?
