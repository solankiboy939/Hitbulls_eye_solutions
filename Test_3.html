Autoencoder Types

  • • Undercomplete: Hidden layer < input layer
  • • Overcomplete: Hidden layer > input layer
  • • Denoising: Trained with corrupted inputs
  • • Sparse: Enforces sparsity in hidden units
  • • Contractive: Penalizes sensitivity to input

Key Concepts

  • • Reconstruction error for anomaly detection
  • • Sigmoid + BCE for binary data
  • • Linear + MSE for real-valued data
  • • PCA = Linear Autoencoder with MSE loss
  • • Nonlinear manifolds require nonlinear AE
